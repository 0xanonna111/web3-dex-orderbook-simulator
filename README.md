# Web3 DEX Orderbook Simulator

A professional Web3 reference project that demonstrates how decentralized exchange orderbook systems can be modeled using clean and modular Web3 architecture.

This repository focuses on simulating buy and sell orders, matching trades, tracking market depth, and generating trading analytics in a beginner-friendly and production-style format.

---

## Supported Networks

- Ethereum  
- Base  
- Polygon  
- Arbitrum  

---

## Project Goals

- Simulate buy and sell orders  
- Match trades using simple logic  
- Track market depth  
- Analyze trading activity  
- Teach DEX trading architecture  

---

## Architecture Overview

This project uses a root-only structure where each file handles a single responsibility such as network access, order registry, matching engine, depth analyzer, and analytics reporting.

---

## Features

- Multi-chain RPC support  
- Orderbook simulation engine  
- Trade matching logic  
- Market depth analytics  
- Clean console-based reporting
